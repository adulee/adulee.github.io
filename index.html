Ini Adalah Biodata ABDULLAH MARWAN HADID

Pria asal Bandung yang tampan dan berani

BANDUNG, RUMAH SAKIT, 05 Maret 2003 MASUK

Cerita Singkat

Abdullah Marwan Hadid, itulah nama saya. Saya biasa di panggil Hadid,Dulee.

Saya lahir di Bandung pada 05 Maret 2003. Saat saya lahir, ayah saya tidak ada disana karena dia sedang kerja di luar kota tapi, kata kakek saya, saya di temukan di pasar ketika ibu saya sedang belanja.

Pendidikan

ketika saya mulai tumbuh, saya memasuki taman kanak-kanak peril, cikadut, bandung, jawa barat. disana masa kanak kanak ku di bentuk banyak menemukan teman bermain yang seru dan baik hati

SMP PGII Bandung, di sinilah saya melanjutkan studi saya, di jenjang smp ini sebetulnya saya tidak ada pengalaman yang bermanfaat, hanya saja bagi saya yang berkesan adalah pertemanan yang seruu.

SMAN 10 Bandung, setelah mendapat nilai bagus pada UN smp saya beruntung mendapatkan bangku di SMAN 10 Bandung. Di sinilah pengalaman saya bertambah mulai dari pertemanan, percintaan, organisasi dan manajemen waktu. disini juga saya belajar banyak hal diluar akademik seperti bekerja sama,memahami cara berteman, penempatan diri pada lingkungan, penyelesaian masalah dan softskill lainya

Setelah senang senang di jenjang pendidikan wajib saya melanjutkan studi di Institut Teknologi Nasional. Saya mengambil prodi Informatika karena menurut saya prodi ini sangat menjanjikan untuk pekerjaan kedepannya. Sekarang saya sudah memasuki semester 3, sejujurnya saya sangat nyaman dan semangat, meskipun sedikit sibuk untuk menjalani tugas matakuliah dan himpunan, tapi seruu.

Cita-Cita

Jika bicara cita-cita, sejujurnya cita-cita saya berganti ganti seiring berjalanya waktu, ketika kecil saya ingin menjadi profesor dan tentara, setelah remaja cita-cita saya berubah dan ingin menjadi Agen Intelejen, saya ingin masuk Sekolah Tinggi Intelejen Negara, namun pada saat itu saya merasa kurang persiapan khususnya di fisik. setelah sekarang menduduki bangku kuliah cita-cita saya ada 2, ingin menjadi cyber security atau orang IT yang bekerja di perusahaan seni (multimedia), namun pada akhirnya cita-cita saya hanya 1, bisa kebeli rumah dan ada kendaraan.

The dreams of yesterday are the hopes of today.

Pengalaman Singkat

To go places and do things that have never been done before – that’s what living is all about.

Bicara tentang pengalaman, bisa dibilang pengalaman saya belum terlalu banyak. Saya pertama kali masuk organisasi ketika menduduki bangku SMA disana saya belajar banyak, tapi menurut saya masih banyak kekurangan dalam organisasi tersebut, bisa dibilang prokernya tidak jelas, dan isi organisasinya pun kurang mengerti posisi mereka. Selain di kuliah, saya pun pernah mengikuti bela negara di SCAPA bandung selama 2 hari.
